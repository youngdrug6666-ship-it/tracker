📖 Руководство по трекеру боя D&D 5e

  • ➕ Добавление участников: Нажмите «📖 Бестиарий» — откроется панель со списком монстров. Используйте фильтры по папкам и CR. Кнопка «📄 Инфо» показывает полную карточку монстра со статами, действиями и заклинаниями. «➕ Добавить» помещает монстра в бой. Героев можно добавить кнопкой «🛡️ Новый герой» (вручную) или импортировать JSON-файл персонажа (Long Story Short) через «📂 Импорт героя».
  • 🎲 Инициатива: В карточке есть поле «Инициатива» и кнопка «🎲 d20» — бросок 1d20 + бонус инициативы. Кнопка «🎲 Инициатива монстрам» бросает инициативу всем монстрам сразу. После ввода значений нажмите «⚡ Сортировать по иниц» — порядок ходов обновится.
  • ❤️ Хиты и урон: Поле ввода для урона/лечения и кнопки: ⚔️ — нанести урон, ½ / ¼ / ×2 — модификаторы, 💚 — лечение. У монстров есть кнопка «❤️🎲» — перебросить ХП по формуле.
  • 📋 Состояния: Нажмите «📋 Состояния», чтобы открыть панель выбора. Можно выбрать несколько состояний — они отобразятся цветными бейджами. Чтобы снять состояние, нажмите на его бейдж или снова выберите в панели. Кнопка «❌ Сбросить всё» убирает все состояния сразу.
  • 🔮 Заклинания: У монстров-заклинателей отображаются ячейки заклинаний (используйте +/-). В детальном окне (через «☰») вы увидите полный список заклинаний существа, сгруппированный по уровням. Клик по названию заклинания открывает его полное описание. В тексте описания заклинания подсвечиваются и тоже кликабельны.
  • 💾 Кампании (сессии): В панели инструментов есть выпадающий список с названием текущей кампании. Вы можете переименовывать, создавать новые и удалять кампании. Всё состояние боя автоматически сохраняется в браузере. Кнопка «🔄 Сброс сессий» удаляет все данные и начинает новую чистую кампанию.
  • ⌨️ Горячие клавиши: Стрелки ← / → переключают ход (предыдущий / следующий). Esc закрывает модальные окна.
  • 📱 Интерфейс: Верхнюю панель и панель инструментов можно свернуть кнопками справа (▲). Бестиарий открывается слева и сдвигает поле боя. Карточки можно перетаскивать для ручной сортировки.
